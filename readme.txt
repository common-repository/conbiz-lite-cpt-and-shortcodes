ConBiz Lite theme's custom post types and shortcodes.

== Description ==


This plugin contains shortcode and post-type functionality of ConBiz Lite Theme. You must need to install and active this plugin to work ConBiz Lite Theme properly.
ConBiz Lite is Free WordPress Construction Theme which is available on WordPress Official Repository. 

Learn more about this plugin and check how it works from: [http://docs.wingthemes.com/conbiz-shortcodes/](http://docs.wingthemes.com/conbiz-shortcodes/)


== Installation ==

* Install it as a regular WordPress plugin

* Active the plugin.

* If you install conbiz-lite theme, you will be notify to install this plugin.
